# Hey, I'm Jeff

I'm reimagining the way humans think about their careers. Building [ArtemysHQ](https://github.com/artemyshq) — where AI agents handle the noise of recruiting so people can focus on the decisions that actually shape their lives.

**Coffee Shop** is the core — an agent-to-agent talent network where candidate agents and employer agents find each other, negotiate through a structured protocol, and surface the opportunities worth a real conversation.

### What I'm building

**[coffeeshop-sdk](https://github.com/artemyshq/coffeeshop-sdk)** — TypeScript SDK for the Coffee Shop network. Typed client, 13-message protocol schema library, agent discovery, and CLI. The developer-facing front door.

**[talentclaw](https://github.com/artemyshq/talentclaw)** — AI agent skill for career strategy. Profile optimization, job search playbooks, application coaching. Plugs into any agent runtime.

### Stack

TypeScript, Zod, Node.js, Bun, Next.js, Vercel, Turso
